# Yemekhane App

Yemekhane App, kampüs yemekhane menülerinin mobil ortamda görüntülenmesini amaçlayan Flutter tabanlı bir mobil uygulamadır.
Uygulama, kullanıcıların günlük yemek menülerine hızlı ve pratik şekilde erişebilmesini hedefler.

Bu proje Flutter kullanılarak geliştirilmiş olup Android ve iOS başta olmak üzere çoklu platform desteğine sahiptir.

---

## Özellikler

* Flutter ile geliştirilmiş çapraz platform mobil uygulama
* Android, iOS, Web ve masaüstü platform desteği
* Basit ve anlaşılır kullanıcı arayüzü
* Menü içeriklerinin görsel destekli sunumu
* Asset (resim) yönetimi ile zenginleştirilmiş UI

---

## Proje Yapısı

```
├── android/        # Android platform dosyaları
├── ios/            # iOS platform dosyaları
├── web/            # Web platform dosyaları
├── linux/          # Linux masaüstü yapı
├── macos/          # macOS masaüstü yapı
├── windows/        # Windows masaüstü yapı
├── lib/            # Dart kaynak kodları
├── assets/         # Görseller ve medya dosyaları
├── test/           # Test dosyaları
├── pubspec.yaml    # Proje bağımlılıkları ve ayarlar
└── README.md       # Proje dokümantasyonu
```

---

## Kurulum ve Çalıştırma

### Gereksinimler

* Flutter SDK
* Android Studio veya VS Code
* (iOS için) macOS ve Xcode

---

### Kurulum Adımları

1. Depoyu klonlayın:

```bash
git clone https://github.com/isil-ada/yemekhane_app.git
cd yemekhane_app
```

2. Gerekli bağımlılıkları yükleyin:

```bash
flutter pub get
```

3. Bağlı cihazları listeleyin:

```bash
flutter devices
```

4. Uygulamayı çalıştırın:

```bash
flutter run
```
